About Credit Card Fraud Detection

Our Mission

Our mission is to provide a reliable system for detecting fraudulent credit card transactions, protecting businesses and customers from financial losses through advanced machine learning.

How It Works

Our system uses Random Forest classification to analyze transaction data and identify fraud patterns. The model processes 30 features transformed with PCA to protect sensitive information while delivering accurate results in milliseconds.

Technology

Built using Python with NumPy and Pandas for data processing, Scikit-learn for machine learning, and Flask with HTML/CSS for the web interface. All components are optimized for speed and accuracy.

How to Use

Click the "Check Fraud" button and enter 30 comma-separated values (V1-V28, Amount, Time). The system will instantly analyze the data and determine if the transaction is fraudulent or normal.
